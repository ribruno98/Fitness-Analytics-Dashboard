# Painel de Análise de Fitness

## Visão Geral

Este projeto é um painel interativo de Power BI desenvolvido para um ginásio. Fornece informações sobre o desempenho dos membros, instrutores, treinos e medidas corporais.

## Ferramentas Utilizadas

- Power BI
- DAX
- Power Query
- Excel

## Páginas do Painel

- Painel Executivo
- Perfil do Cliente
- Desempenho do Instrutor
- Análise de Treinos

## Recursos

- Filtros interativos
- Cartões KPIs
- Análise de tendências
- Desempenho dos treinos
- Comparação de instrutores
- Acompanhamento do progresso do cliente

## Conjunto de Dados

Conjunto de dados sintético criado para fins de portfólio.
